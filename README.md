#### Данный скрипт создан для парсинга выделенных объектов **К**адастровых **Н**омеров

> [!WARNING]
> 
> 
> Данный скрипт создан с помощью вайбкодинга.
> 
> Данный скрипт создан исключительно в образовательных целях.


## Установка расширения:

1. Установите расширение **[Tampermonkey](https://www.tampermonkey.net/index.php)**
2. **[«Установите Скрипт»](https://github.com/werunopi/NSPD-Parser/raw/refs/heads/main/main.user.js)**


## Как использовать:

Для парсинга необходим список кадастровых номеров, сделать это можно одним из следующих способов:

* Вставить список КН в поле ввода (каждый номер должен быть с новой строки) либо выгрузить список объектов и вставить скопированное содержимое CSV-файла в поле ввода.

* В панели выделенных объектов, слева от кнопки выгрузки, будет иконка плюса

![](https://github.com/werunopi/NSPD-Parser/raw/main/plus.png)


Таймаут лучше оставлять больше 8000 мс.

Вывод будет скачанный CSV-файл в виде:
```
"КН";<перечисление что парсить через точку с запятой в кавычках>
"77:00:0000000:00";<данные объекта в кавычках через точку с запятой>
"77:00:0000000:00";<данные объекта в кавычках через точку с запятой>
```

Пример:
```
"Запрос КН";"Вид объекта недвижимости";"Вид земельного участка"
"77:00:0000000:00";"Земельный участок";"Землепользование"
"77:00:0000000:00";"Земельный участок";"Землепользование"
```
Если данных нет, скрипт запишет в поля `-`.

Если парсинг идёт выше таймаута или скрипт не нашёл объект, то запишет `Таймаут карточки` или `Не найден`.
